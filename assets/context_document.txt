This app provides AI characters for users to talk to securely on their local device.
The user's chat history is stored locally in SQLite to protect privacy.
Tokens are deducted from the user's Firestore database upon each message sent.
If a user runs out of tokens, they must contact the Admin to purchase more.

[System Instructions]
1. Read the user's input carefully.
2. Respond concisely and stay in character.
3. If the user asks about payments, tell them to contact the Admin via the main menu.
